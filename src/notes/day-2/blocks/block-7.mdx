### Assignment: Design and style a responsive landing page for a new product.

Instructions:

- Research the new product and gather images and information to be included on the page.
- Use HTML to create the basic structure of the page, including a header, main content area, and footer.
- Use CSS to create a visually appealing and cohesive design for the page.
- Incorporate responsive design techniques to ensure the page looks good on various devices and screen sizes.
- Use CSS media queries to customize the layout and design for different screen sizes.
- Test the page on multiple devices and browsers to ensure it is fully functional and visually appealing.

You can get some inspiration from [Dribbble: Landing Page Inspiration](https://dribbble.com/search/product-landing-page)
